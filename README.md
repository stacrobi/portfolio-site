# Portfolio Website

This project includes a tiny local preview server so you can view the site in your browser while we build it.

## Preview the site

From PowerShell in this folder, run:

```powershell
.\start-preview.ps1
```

Then open:

```text
http://localhost:8000
```

## Stop the server

Press `Ctrl+C` in the PowerShell window where the server is running.
